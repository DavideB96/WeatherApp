# WeatherApp

A simple weather application built with HTML, CSS and Vanilla JavaScript.

## Features
- Fetches real-time weather data from OpenWeather API
- Loading state animation
- Error handling
- LocalStorage persistence (last searched city)
- Dynamic weather icons (emoji-based)

## Tech Stack
- HTML
- CSS
- JavaScript (ES6)
- OpenWeather API

## Note
This project uses a public demo API key for educational purposes.  
In a production environment, API keys and secrets should be handled server-side.
