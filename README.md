# 🌦️ Weather Forecast App

A modern and responsive web application that provides real-time weather data for any city worldwide.

## 🚀 Demo

👉 Enter a city name to instantly view:

- Current temperature  
- Feels like temperature  
- Humidity  
- Wind speed and direction

## 🛠️ Tech Stack

- HTML5  
- CSS3 (modern UI & animations)  
- JavaScript (Vanilla)  
- OpenWeatherMap API  

---

## ⚙️ Features

- 🔍 City-based weather search  
- 🌡️ Real-time weather data  
- 📅 5-day forecast  
- 📍 Automatic geolocation  
- 💡 Autocomplete search suggestions  
- 🎨 Interactive and modern UI  
- ⚡ Loading state handling  
- ❌ Error handling (invalid city / network issues)  
- 💾 Last searched city saved via localStorage  
- 🌍 Multi-language support (Italian)
